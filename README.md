# projet-S4-

Pour faire fonctionner ce projet il faudra:

Installation et Préparation:
- Avoir la version 7.4 de php ou une version plus récente.
- cloner le répertoire depuis votre terminal avec la commande: git clone https://github.com/xFireANTx/projet-S4-/
- Se placer dans le dossier "projet-S4-" avec la commande: cd projet-S4-
- Pour être sûr que le php puisse écrire dans la base de données, entrer la commande: chmod 777 utilisateurs.json

Accès au site:
- Depuis le dossier "projet-S4-" (la racine) entrer la commande: php -S localhost:8000 (si 8000 ne fonctionne pas essayer 8080)
- Pour accéder à l'expérience Utilisateur, entrer dans la barre de recherche du navigateur: localhost:8000/utilisateur/Accueil.html
- Pour accéder à l'expérience Administrateur, entrer dans la barre de recherche du navigateur: localhost:8000/administrateur/admin.html
