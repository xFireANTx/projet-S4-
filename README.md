# projet-S4-

Pour faire fonctionner ce projet il faudra:

Installation et Préparation:
- Avoir la version 7.4 de php ou une version plus récente.
- cloner le répertoire depuis votre terminal avec la commande: git clone https://github.com/xFireANTx/projet-S4-/
- Se placer dans le dossier "projet-S4-" avec la commande: cd projet-S4-
- Pour être sûr que le php puisse écrire dans la base de données, entrer la commande: chmod 777 utilisateurs.json

Accès au site:
- Depuis le dossier "projet-S4-" (la racine) entrer la commande: php -S localhost:8000 (si 8000 ne fonctionne pas essayer 8080)
- Entrer dans la barre de recherche du navigateur: localhost:8000/utilisateur/Accueil.html
- Pour accéder à l'expérience Utilisateur, créer un nouvel utilisateur depuis la page incription puis connectez vous avec connexion
- Pour accéder à l'expérience Administrateur, connectez-vous avec le mail: admin@japindien.com et le mot de passe: "12345678"
- Pour accéder à l'expérience restaurateur (accéder aux commandes), connectez vous avec le mail: restaurateur@mail.fr et le mot de passe: "22222222" (8X2)
- Pour accéder à l'expérience livreur (accéder aux livraisons), connectez vous avec le mail: livreur@mail.fr et le mot de passe: "11111111" (8X1)
